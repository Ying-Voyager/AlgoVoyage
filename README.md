# 🚀 个人算法与工程框架 (My Algorithm Framework)

## 📖 项目简介
本项目是用于沉淀个人核心算法、存放各类数学建模比赛代码以及科研项目仿真的综合性代码库。旨在通过高内聚、低耦合的工程化标准，实现代码的高效复用与迭代。

## 📂 目录结构说明

* **`core/`**：核心算法库（如 TimesNet 时序模型、动力学仿真模型等底层逻辑）。
* **`utils/`**：通用工具箱（如统一的数据读取接口、Matplotlib 标准化画图/LaTeX渲染工具等）。
* **`projects/`**：科研与工程项目（如仿生蝴蝶机构仿真、磁脉冲焊接数据处理等）。
* **`competitions/`**：历次竞赛代码（如 MCM/ICM 建模代码等，独立运行环境）。

## 🛠️ 环境与依赖
* **Python 版本**: >= 3.8
* **核心依赖库**: 
  * `numpy`
  * `matplotlib` (图表默认使用 LaTeX 渲染格式)
  * `pandas`

## ⚙️ 开发规范
1. 所有核心算法均采用面向对象 (OOP) 封装。
2. 内部路径全部使用相对路径，确保代码可移植性。
3. 纯英文命名文件与目录，代码内部保留详尽的中文注释。
