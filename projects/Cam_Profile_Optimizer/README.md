## 1. 问题背景

### 机构参数

| 参数 | 数值 |
|------|------|
| 运动规律 | 正弦加速度（推程 & 回程） |
| 推程角 φ_push | 90° |
| 远休角 φ_far | 60° |
| 回程角 φ_return | 90° |
| 近休角 φ_near | 120° |
| 行程 h | 15 mm |
| 滚子半径 r_r | 4.5 mm |

### 研究目标

对于上述确定运动规律的凸轮机构，以基圆半径 r₀ ∈ [30:5:90] mm、偏置量 e ∈ [-15:3:15] mm 为参数空间，计算各参数组合下**推程最大压力角 α_max** 的分布规律，并进一步求解：

- **任务一**：用三维曲面图直观展示 α_max 随 r₀ 与 e 变化的全貌；
- **任务二**：筛选出满足许用压力角 α_max ≤ 30° 的可用参数区域；
- **任务三**：对任意给定的 r₀，求使 α_max 最小的**最优偏置量 e_opt**。

## 2. 数学模型

### 推杆位移方程（正弦加速度，完整四阶段）

代入本题参数（h = 15 mm，φ_push = π/2，φ_far = π/3，φ_return = π/2，φ_near = 2π/3），从动件全周位移方程为：

$$s(\theta) = \begin{cases} -\dfrac{15}{2\pi}\sin(4\theta) + \dfrac{30\theta}{\pi}, & 0 \leq \theta < \dfrac{\pi}{2} \\[10pt] 15, & \dfrac{\pi}{2} \leq \theta < \dfrac{5\pi}{6} \\[10pt] 15 - \dfrac{30}{\pi}\!\left(\theta - \dfrac{5\pi}{6}\right) + \dfrac{15}{2\pi}\sin\!\left(4\theta + \dfrac{2\pi}{3}\right), & \dfrac{5\pi}{6} \leq \theta < \dfrac{4\pi}{3} \\[10pt] 0, & \dfrac{4\pi}{3} \leq \theta < 2\pi \end{cases}$$

对凸轮转角的导数（等效速度，仅推程段用于压力角计算）：

$$\frac{ds}{d\theta} = \frac{h}{\varphi}\left(1 - \cos\frac{2\pi\theta}{\varphi}\right) = \frac{30}{\pi}(1 - \cos 4\theta), \quad 0 \leq \theta < \frac{\pi}{2}$$

### 压力角公式

偏置直动滚子推杆的推程压力角满足：

$$\tan(\alpha) = \frac{\dfrac{ds}{d\theta} - e}{\sqrt{r_0^2 - e^2} + s(\theta)}$$

其中 e 为偏置量（推杆轴线与凸轮转动中心的距离，向左为正）。

## 4. 各脚本功能说明

### 4.1 `Cam_visualization.m` — 单参数组可视化

**功能：** 在给定 r₀ 和 e 下，完整绘制凸轮机构的运动学特性与轮廓线。

输出图形：

| 图 | 内容 |
|----|------|
| 图1（三子图） | 从动件位移曲线 / 速度曲线 / 压力角曲线（四阶段分色，含许用压力角参考线） |
| 图2 | 凸轮轮廓线（基圆、偏置圆、理论廓线四阶段分色、实际廓线、滚子位置） |

**使用方式：** 修改脚本开头的 `r_0`、`e`、`r_roller` 参数后直接运行。

---

### 4.2 `Cam_3d_analysis.m` — 全参数空间三维分析

**功能：** 遍历 r₀ ∈ [30:5:90]、e ∈ [-15:3:15] 的全部参数组合，计算每组的推程最大压力角，生成深色主题三维曲面图。

输出图形：

| 图 | 内容 |
|----|------|
| 图1 | 全部有效参数的 α_max 三维曲面（含许用压力角 30° 半透明参考平面） |
| 图2 | 过滤后仅保留 α_max ≤ 30° 的可用参数区域曲面（含边界轮廓线投影） |

控制台同时输出有效组合数、满足许用条件的组合数及覆盖率统计。

---

### 4.3 `Cam_triple_analysis.m` — 三图联合精细分析

**功能：** 在粗扫（步长 5mm）的基础上进行细密扫描（e 步长 0.1 mm），联合输出三张分析图。

输出图形：

| 图 | 内容 |
|----|------|
| 图1 | 固定 r₀=30 mm，α_max 随偏置量 e（步长 1 mm）的变化曲线（渐变色，含最优点标注） |
| 图2 | 不同 r₀ 下的最优偏置量 e_opt 散点图（细扫连续曲线 + 粗扫对比 + 二次多项式拟合） |
| 图3 | 固定 r₀=30 mm，α_max 与 α_mean 的双纵轴对比图（揭示两者最优偏距不重合的现象） |

控制台同时输出各 r₀ 对应的 e_opt（细扫）、e_opt（粗扫）及对应最小 α_max 的详细数据表。

---

### 4.4 `Cam_optimal_e.m` — 交互式最优偏距求解器

**功能：** 交互式输入任意基圆半径 r₀，调用 `fmincon` 数值优化算法，在约束 |e| < r₀ 的范围内求解使推程最大压力角最小的最优偏置量 e_opt。

**运行示例：**
```
请输入基圆半径 r0 (mm): 30

========== 优化结果 ==========
基圆半径   r0      = 30.0000 mm
最优偏距   e_opt   = 8.4941 mm
最小最大压力角     = 17.4424 °
==============================
```

## 5. 核心结论

### 5.1 α_max 的分布规律

- 在 r₀ ∈ [30, 90] mm、e ∈ [-15, 15] mm 的参数空间内，α_max 呈现出明显的非单调曲面形态。
- **增大 r₀**：α_max 单调减小，基圆半径越大，压力角越小，传动性能越好。
- **改变 e**：对于任意固定的 r₀，α_max 随 e 先减小后增大，存在唯一的最优偏置量 e_opt，使 α_max 取得最小值。
- 负偏置（e < 0）通常使压力角增大，正偏置在适当范围内可显著降低压力角。

### 5.2 可用参数区域（α_max ≤ 30°）

在题目给定的离散参数网格（r₀ 步长 5 mm，e 步长 3 mm）下：

- 满足许用压力角 ≤ 30° 的参数组合占有效组合的约 **60%**。
- 可用区域边界在 r₀–e 平面上近似呈线性关系，拟合直线方程约为：

$$e \approx -0.5275 \cdot r_0 + 13.7093$$

即：基圆半径越小，要满足许用压力角条件所允许的最大正偏置量越小。

### 5.3 最优偏距的规律

- 对于 r₀ ∈ [30, 90] mm 的全部取值，最优偏置量 e_opt 约在 **8.5 ~ 9.5 mm** 之间单调缓慢增大。
- 当 r₀ → ∞ 时，e_opt 趋近于理论极限值 ≈ **9.5 mm**（与 h、φ 等运动参数有关，与 r₀ 无关）。
- 在粗扫（步长 3 mm）下，全部 r₀ 的最优 e 均落在 **e = 9 mm**；细化到步长 0.1 mm 后，数值优化给出 e_opt ≈ **8.49 mm**（r₀=30）。

### 5.4 最大压力角与平均压力角的差异

- α_max 最小时对应的偏置量（e ≈ 9 mm）与 α_mean 最小时对应的偏置量（e ≈ 11 mm）**并不重合**。
- 实际设计中可根据需求赋予两者不同权重：**侧重安全性**（防止卡死）则优先最小化 α_max；**侧重传动效率**（降低平均摩擦损耗）则优先最小化 α_mean。
- 综合加权指标（压力角优化率 η）可兼顾两者，η 值越大代表机构性能越优。
